Item Description
Network Classifications
  • LAN
  • MAN
  • WAN
  • local area network, connects machines within the same building(s) as in a school campus or a manufacturing plant
  • metropolitan area network, intermediate size like a local community
  • wide area network, connects machines across great distances, ex: opposite sides of the world
Network models
  • Bus network
  • star network
 The pattern in which machines are connected.
  • Machines are connected to a common communication line called a bus.
  • One machine seves as a central focal point to which all the others are connected,the access point (AP).
Protocols
  • CSMA/CD Ethernet
  • CSMA/CA Wifi
  • Used by bus networks to control the right to transmit messages. Carrier Sense, Multiple Access with Collision Detection
  • Used to address the hidden terminal problem(ie, machines can't hear eachother but can talk to the AP) of STAR networks, prioritizing collision avoidance rather than detection. Carrier Sense, Multiple Access with Collision Avoidance.
Combining networks:
  • repeater
  • bridge
  • switch
  • router
  • gateway
  • a device that passes signals (usually with an amplifier) between two original buses without considering the meaning of the signal
  • a bridge connects two buses but tracks the intended address of the signals being recieved and only transmits them if the recieving address is on the other side of the bridge, making communication more efficient.
  • a bridge with several connections rather than just two.
  • a special purpose computer used for forwarding messages between networks on an internet.
  • the point at which one network is linked to an internet. ex: router, or AP/router unit
Client/server model
  • P2P: peer-to-peer
  • distributed system
  • cloud computing
 A popular convention used for interprocess communication identifying a client(makes requests) and a server(executes requests)
  • an alternative interprocess communication involving processes that provide service to and recieve service from each other, ie. instant messaging
  • software units that execute as processes on different computers collectively as one large unit.
  • a distributed system utilizing the network computing power of the internet and allowing for servers to perform tasks without the knowlege of where the hardware machine is located. (concerns abt privacy vs scalability)
Internet Addressing
  • Tier 1
  • Tier 2
  • Tier 3
 ISPs (internet service providers) connect and maintain what we think of as the internet
  • very high speed, high capacity, international WANs. (backbone of internet)
  • connected to teir 1s, regional, less potent in scope
  • AkA access ISP, are an independant internet(intranet) opperated by a single authority that is supplying internet to individual homes/buisnesses. ex: cable
IP address
  • ICANN
  • DNS
  • TLD: Top-level domain
  • Country-code TLDs
 An internet-wide addressing system that assigns a unique identifying address to each computer in the system
  • Internet Corporation for Assigned Names and Numbers, a non-profit corporation established to coordiante the internet's operation. assigns blocks of IP addresses to ISPs to distribute.
  • Domain name system, a series of servers that perform the role of directory from mnemonic names(URLs) into IP addresses for the clients to find their destinations.
  • suffixes such as edu or com that reflect a domain's broader classification.
  • two letter TDLs for each specific country expanded outside of the US
Internet protocols
  • FTP
  • HTTP
  • SMTP
 Rules for applications to follow
  • File Transfer Protocol, used for listing and copying files over a network
  • Hyper Text Transfer Protocol, most commonly used protocol, follows instructions from webpages
  • Simple Mail Transfer Protocol, defines how two computers on a network may interact when transmitting email.
World Wide Web
  • URL
  • client-side activities
  • server-side activities
 A hypertext document format for embedding hyperlinks to other documents, a protocol for transferring hypertext across the network, and a server process that supplied hypertext pages upon request.
  • Uniform Reasource Locator, a uique address used to locate and retrieve documents
  • activities performed by a client on either a browser or webserver such that is additional to the displaying of a website
  • Activities performed by a server on either a browser or webserver such that is additional to the displaying of a website, ie animation
Internet four-layer software
  • application layer
  • transport layer
  • network layer
  • link layer
 The software for controling communication over the internet.
  • consits of those software units such as clients and servers that use internet communication to carry out their tasks.
  • provides an address for the message and splits it into smaller segements for transfer(packets)
  • maintains the router's forwarding table, and uses it to direct packets
  • incharge of recieving and transmitting packets.
Internet is:
  • packet-switching network
  • fault tolerant
  • redundant
  • a computer network that breaks data into packets and sends them through switches to their destination.
  • the ability of a system to continue operating when one or more of its components fail
  • having multiple pathways for data transmission, so if one path fails, data can still flow through alternative routes.
Cybersecurity
  • malware
  • virus
  • worm
  • Trojan horse
  • spyware
  • phishing
  • DDoS attack
  • malicious software
  • software that infects a computer by inserting itself into programs that aleady reside in the machine
  • an autonomous program that transfers itself though a network, taking up residence in computers and forwarding copies of itself to other computers.
  • a program that enters a computer system disguised as a desirable program that is willingly imported by the victim. Once in the computer, it performs additional activities that might have harmful effects.
  • or sniffing software, is software that collects information about activities at the computer on which it resides and reports it back to the instigator of the attack.
  • a technique of obtaining information by asking for it through posing as an authority usually via email.
  • distributed denial of service, the process of overloading a computer with messages, distributed refers to the activation of unsuspecting computers that generate the messages from many different hosts, making it harder to deter.
Protection
  • firewall
  • spam filter
  • proxy server
  • antivirus software
  • cryptography
  • public key encryption
  • a primary prevention technique that filters traffic passing through a point in the network, allowing them to block outgoing messages with certain destination addresses, or block incoming messages from origins that are known to be sources of trouble.
  • a firewall designed for the specific purpose of blocking unwanted email.
  • a software unit that acts as an intermediary between a client and server with the goal of sheilding the client from adverse actionsof the server.
  • used to detect and remove the presence of known viruses and other infections.
  • the study of sending a recieving messages securely in the presence of adversaries.
  • techniques by which the encryption and decryption algorithms are known, and one of the keys is known, but the encoded messages remain secure.
