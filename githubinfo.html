Questions Answers
What is a version control system? A version control system (VCS) is a software tool that tracks and manages changes made to files over time.
What is Git? Git is a free open-source VCS that allows users to track code changes, store code, and revert to previous versions.
What is Github? Github is a web-based platform that allows users to store, share, and collaborate on code.
What is the difference between Git and Github? Git is a VCS that allows developers to track changes in their code. Github is a web-based hosting service for git repositories. Github requires Git to run.
Who started Github and how was it started? Github was developed by Chris Wanstrath, P. J. Hyett, Tom Preston-Werner, and Scott Chacon using Ruby on Rails, and the service was launched in February 2008.
What company owns it now? Microsoft purchased GitHub on October 26, 2018.
How much does a Github account cost? It is free to operate a personal account but there are paid plans with additional features.
What is the Octocat? The Octocat is GitHub's mascot

Git/Github terms:

Repository
A place to store files, code, and their revision history on the cloud
Commit
A commit records changes to one or more files in your branch
Fork
A fork is a new repository that shares code and visibility settings with the original "upstream" repository.
Push
The git push command is used to upload repository content to a remote repository. Pushing is how you transfer commits from your local repository to a remote repository.
Pull Requests
A pull request (PR) is a proposal to merge changes from one branch into another branch.
Wokflows
Workflows are configurable, automated processes that can run one or more jobs to perfom tasks in a repository.
Issues
Issues is a tool that helps users track and manage work by creating, discussing, and planning items in a repository.
Raw Button
You can view or copy the raw content of a file without any styling.
Blame Button
Is used to examine the contents of a file line by line and can see when each line was last modified and who the author of the modifications was.
